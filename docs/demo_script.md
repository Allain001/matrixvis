# LineaVis 答辩演示视频脚本（约 150s）

## 0-10s 开场
画面：主页 Hero + 3D 背景缓慢旋转  
字幕：LineaVis 线性代数可视化平台

## 10-35s 创新亮点
画面：滚动到“创新与技术深度”板块  
操作：依次悬停四个创新卡片  
字幕：操作—计算—验证闭环

## 35-60s 可视化核心
画面：进入“矩阵可视化”  
操作：修改矩阵元素，显示变换效果  
字幕：矩阵作用于基向量与单位方格

## 60-90s 实验室挑战
画面：进入“实验室”  
操作：展示行列式、迹、特征值  
操作：完成“面积加倍”和“纯旋转”任务  
字幕：任务驱动验证理解

## 90-110s 结果导出
画面：点击“导出分析结果”  
字幕：一键导出矩阵分析结果

## 110-140s 学习路径与应用价值
画面：滚动到学习模块  
字幕：系统化学习路径与应用场景

## 140-150s 收尾
画面：回到顶部或 CTA  
字幕：让线性代数看得见、拖得动、可验证
