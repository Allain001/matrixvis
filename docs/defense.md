# LineaVis 线性代数可视化平台

## 作品定位
LineaVis 面向高校线性代数教学与竞赛评审展示，提供“可操作、可验证、可解释”的矩阵变换实验平台。

## 创新点
1. 交互式 3D 场景将抽象矩阵变换转化为可视化实验。
2. 以行列式、迹、特征值等不变量驱动结果验证，形成“操作—计算—验证”的闭环。
3. 任务挑战机制引导学生构造满足性质的矩阵，强化理解而非记忆。
4. WebGL 容错渲染与 Streamlit 一键部署，适配线上答辩与教学场景。

## 技术深度
1. 代数层：实时计算行列式、迹、特征值与正交性判别。
2. 几何层：矩阵作用于基向量与单位方格，展示旋转、剪切、缩放与反射。
3. 渲染层：Three.js + React Three Fiber 实时渲染，支持交互旋转与缩放。
4. 工程层：模块化架构、前端构建产物与 Streamlit 集成部署。

## 核心功能
1. 矩阵编辑与实时变换可视化。
2. 不变量分析与变换类型判别。
3. 任务挑战与达成状态提示。
4. 学习模块路径与场景化介绍。
5. 一键导出矩阵分析结果（JSON）。

## 作品运行方式
1. 在线部署：Streamlit Cloud（提交后的公开链接）。
2. 本地运行：`npm run dev` 或 `streamlit run streamlit_app.py`。

## 应用价值
1. 教学：降低抽象难度，提高课堂互动与理解效率。
2. 竞赛：评审可现场构造矩阵并验证结果，提升可信度。
3. 传播：可扩展至机器学习、图形学与数据可视化课程。

## 后续规划
1. 三维矩阵与特征分解（SVD、PCA）可视化。
2. 学习路径评分与个性化推荐。
3. 教学版与竞赛版主题皮肤切换。
